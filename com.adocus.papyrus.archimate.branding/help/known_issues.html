Known issues

Below are some known issues that at the time for release of this add-in have not been solved. However, we and others within the open source community continually work to improve the add-in.
  • New models need to be "migrated", i.e. the migration dialog is shown even for new models
  • Elements can not be dropped directly to Grouping, they have to be created to the side of the Grouping element and after that "dragged" onto it
  • Some relations do not reflect correct appearence according to ArchiMate specification based upon exceptions in Papyrus when trying to do so
